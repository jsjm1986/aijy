知识图谱记忆格式

数学知识记忆包示例

Knowledge Graph 
{
    "nodes": [
        {
            "id": 1,
            "type": "concept",
            "name": "微积分",
            "attributes": {
                "difficulty": "advanced",
                "importance": "high"
            }
        },
        {
            "id": 2,
            "type": "theorem",
            "name": "牛顿-莱布尼茨公式",
            "attributes": {
                "field": "calculus",
                "applications": ["物理", "工程"]
            }
        }
    ],
    "edges": [
        {
            "source": 1,
            "target": 2,
            "relationship": "contains",
            "weight": 0.8
        }
    ],
    "embeddings": {
        "微积分": [0.1, 0.2, 0.3],
        "导数": [0.2, 0.3, 0.4]
    },
    "context": {
        "domain": "mathematics",
        "prerequisites": ["基础代数", "几何"],
        "learningPath": ["极限", "导数", "积分"]
    }
}

格式说明

  • nodes: 知识节点列表，包含概念、定理等
  • edges: 节点间的关系连接
  • embeddings: 概念的向量表示
  • context: 知识的上下文信息

神经网络记忆格式

钢琴技能记忆包示例

Neural Weights 
{
    "layers": [
        {
            "name": "input",
            "neurons": 1000,
            "activation": "relu"
        },
        {
            "name": "hidden1",
            "neurons": 500,
            "activation": "relu"
        },
        {
            "name": "output",
            "neurons": 100,
            "activation": "sigmoid"
        }
    ],
    "weights": [
        "Float32Array(500000)", // 压缩后的权重数据
        "Float32Array(50000)"   // 压缩后的权重数据
    ],
    "biases": [
        "Float32Array(500)",    // 压缩后的偏置数据
        "Float32Array(100)"     // 压缩后的偏置数据
    ],
    "architecture": {
        "type": "feedforward",
        "inputShape": [1000],
        "outputShape": [100]
    }
}

格式说明

  • layers: 神经网络层的结构定义
  • weights: 网络权重矩阵
  • biases: 偏置值数组
  • architecture: 网络架构信息

经验向量记忆格式

旅行经历记忆包示例

Experience Vectors 
{
    "vectors": [
        "Float32Array(1024)", // 场景1的经验向量
        "Float32Array(1024)", // 场景2的经验向量
        "Float32Array(1024)"  // 场景3的经验向量
    ],
    "timestamps": [
        "2023-01-01T12:00:00Z",
        "2023-01-02T14:30:00Z",
        "2023-01-03T09:15:00Z"
    ],
    "emotions": [
        {
            "joy": 0.8,
            "surprise": 0.6,
            "interest": 0.9
        },
        {
            "joy": 0.7,
            "peace": 0.8,
            "satisfaction": 0.9
        },
        {
            "excitement": 0.9,
            "wonder": 0.8,
            "joy": 0.7
        }
    ],
    "context": {
        "location": "巴黎",
        "weather": "晴朗",
        "companions": ["导游", "游客"],
        "activities": ["参观博物馆", "品尝美食", "街道漫步"]

格式说明

  • vectors: 场景经验的向量表示
  • timestamps: 经验记录的时间戳

  • context: 场景上下文信息
